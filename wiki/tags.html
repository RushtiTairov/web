Тег Предназначение тега
<htmll> Тег содержит внутри себя всю html страницу.
<!DOCTYPE> Объявляет тип документа, также может указывать на версию языка разметки.
<!--...--> Использует для добавления коментариев.
<head> Тег с основными свойствами html страницы.
<body> Тег, в котором находятся все отображаемые на странице элементы.
<title> Тег для определения заголовки страницы.
<link> Одиночный тег для подключения внешних таблиц стилей.
<h> Заголовки шести уровней(по убыванию размера).
<a> Создаёт гипертекстовые ссылки.
<img> Одиночный тег, который вставляет изображение на страницу с помощью атрибута src, значение которого является адрес изображения.
